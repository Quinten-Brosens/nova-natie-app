# Nova Natie App

Interne bedrijfsapp voor Nova Natie (Antwerpse haven-natie: CFS, Forest, Transport, Warehousing).

## Demo openen

Open `Nova-App-Demo.html` in een browser. Op een smartphone: voeg de pagina toe aan het beginscherm voor de native app-ervaring.

De demo doorloopt deze flow:
1. **Login** met Microsoft 365
2. **Afdelingskeuze** — HSEQ volledig uitgewerkt, andere afdelingen binnenkort
3. **HSEQ-module**
   - Onveilige situatie melden (foto · locatie · beschrijving · automatische mail)
   - Inspecties & checklists (OK / NOK / N.v.t. met voortgangsbalk)
   - Mijn meldingen
